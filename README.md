# DATA641-FinalProj

**Problem**: Analysing the quality of a customer support call can be subjective and tedious (Human analysis) as well as tone deaf (automated transcript analysis).

**Solution**: We wish to build a real time, sentiment analysis tool to analyse customer service calls and archive with call summaries.

**Why us**: While similar tools exist, they all rely on audio transcripts (text based) and only analyse after the fact. Our product will take this a step further with real time audio analysis.
